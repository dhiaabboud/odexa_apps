POS Offline Mode

  • 1. Allow Start POS Session (Resume Session) with link POS without Internet and Odoo Server
  • 2. All POS Datas like Products, Customers ... saved on POS Device of POS Clients (PC, Tablet..)
  • 3. When internet come back or Odoo Server is ON, all Orders will sync direct to Odoo Server Backend

Why need this module

  • Odoo Original not allow Resume Session and Starting Sell Product when POS Clients's Internet is Down (Off)
  • Morning each day if Odoo Server down, it mean all POS Clients can not login to Odoo and Starting Session
  • This module allow save POS Session Link and Save All Datas need for POS like products, customers ....
  • And Starting POS Session, Sale Products without Internet and Odoo Server ....
  • All Orders will save Clients's PC (Laptop, Ipad, Mobile ...) and sync to Server when Internet back....
  • All POS Clients can refresh Page, Duplicate POS Link and Sale Products without Internet and Odoo Server....
  • Offline mode automatic active when Internet has problem or Odoo server down (off). No need any manual setting

Guide Install

  • Module have not any required python libraries, applications another depends
  • Only bought module, unzip, copy to your Odoo Addons, Update Module list , Install and Running
